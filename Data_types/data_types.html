DATA TYPES

what type of value we are assigning for variables is called datatypes

Data types are two types

Primitive : simple and immutable(cannot be changed)

NonPrimitive : mutable(can be changed)

    Primitive Types
  • String
  • Number
  • Boolean
  • undefined
  • null
  • symbol
    Non-Primitive Types
  1. Array
  2. Object
  3. Function
